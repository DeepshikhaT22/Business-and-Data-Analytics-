Perform end-to-end analysis: data cleaning → analysis → visualization → insight generation

Excel
SQL
Power BI
Statistics
Tableau
Jupyter Notebook

A portfolio for internships/jobs
A reference for future advanced analytics projects

To build a strong foundation in Business Analytics and maintain a structured professional project portfolio.

Open to collaboration, internships, and entry-level opportunities in Data Analytics / Business Intelligence.
